Using Gov Elements in sketch

Adding the Gov Elements

  • Select Sketch in the menu bar
  • open the prefences menu
  • seltct the libraies tab
  • click add libray
  • navigate to gov elements file you downloaded
  • open the file to add it to your sketch libray

Creating Artboards

  • press the insert icon in the top left corner
  • selct artboard
  • create a custom artboard size of 1020px x 861px
  • save the custom artboard size

Adding Elements

  • press the insert icon in the top left corner
  • go to documents in the symbols section
  • find and select the element you want
  • drag it to the correct postion
  • right click the element and detach it from the symbol
