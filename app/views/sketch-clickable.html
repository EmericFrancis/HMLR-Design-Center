Sketch Clickable Prototypes

To create sharable interactive prototypes in sketch you will need to install a plugin.

What you need?

  • Sketch
  • GOV Elemetnts Kit
  • Craft Plugin for sketch
  • An Invison account

Using Gov Elements in sketch

Installing the Craft Plugin

  • download and open the zip folder
  • drag craft manager into the Applications folder
  • open craft manager an icon will appear in your top menu bar
  • select the icon to open craft manager
  • select install for sketch option
  • open the file to add it to your sketch libray

Creating Artboards

  • press the insert icon in the top left corner
  • selct artboard
  • create a custom artboard size of 1020px x 861px
  • save the custom artboard size

Adding Elements

  • press the insert icon in the top left corner
  • go to documents in the symbols section
  • find and select the element you want
  • drag it to the correct postion
  • right click the element and detach it from the symbol
