Google Slides

You can use google slides as a wireframing tool to do this you need to set a custom page size.

  • Create a new googel slides document
  • Go to file > Page setup
  • Select a custom size and change to pixels
  • Input 1200 x 900 pixels and confirm
  • Add Gov.uk Elemetnts
! Warning Set page size before adding elements if you rescale the page with elements on they will distort.
