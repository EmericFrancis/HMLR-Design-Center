Axure

Axure can be used to create medium fedility clickable prototypes. These can then be hosted on axshare which allows you to share prototypes with your team.

Setting Up Axure

  • Download and install the free trial and request a liscence on MySelfService
  • Download the GDS widget kit for Axure
  • Create a new project
  • Drag and drop required elements from widget kit to your prototype

Sharing your project

  • You will need to be added to the HMLR Axshare group
  • Open the project you want to share in axure
  • Click the blue share logo in the top right hand corner
  • Select New project and change the folder to the Land Registry folder
  • Publish the prototype
